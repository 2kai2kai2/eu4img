EU4img
by 2kai2kai2

Requirements:
Python
PIL

How to use:
Run 'run.bat' or 'run.ps1'
When requested, add directory of files. This can be done by dragging the file into the terminal window
Wait. The final file will appear in the run file directory

Custom Font:
Put the ttf file in the run directory as "FONT.TTF"
The eu4 font only works on Windows

Updating:
Updated flags will be necessary when a new version is released.
